## NTUT Network Status

A project to moniter NTUT network, show the ntut network is alive or dead, and show the line chart of NTUT network speed.

## Progress

 - [x] Back-end
 - [x] Front-end

## How to execute it

 - You should have a `firebaseCredential.json` to record it.
 - run `nns.exe` or `go run .`
